# Projeto Final - NaraHoteis

## Contexto
A NaraHoteis é uma rede de hotéis com 12 unidades no estado do Rio de Janeiro.  
A diretoria percebeu queda de receita e pediu uma consultoria de dados para organizar as bases e apoiar decisões.

---

## Etapas
1. Auditoria e Tratamento  
   - Conferimos as 6 bases (clientes, canais, reservas, tipos de quarto, unidades e funcionários).  
   - Corrigimos erros e padronizamos os dados.  
   - Exportamos arquivos limpos em `.csv`.

2. Análise Estatística  
   - Respondemos perguntas da diretoria com Python.  
   - Fizemos gráficos para entender faturamento, capacidade e avaliações.

3. Banco de Dados  
   - Criamos o banco `narahoteis_db` em MySQL.  
   - Definimos tabelas e relacionamentos.  
   - Fizemos consultas com `JOIN`.

4. Painel Power BI  
   - Conectamos os CSVs limpos.  
   - Criamos indicadores de performance.  
   - Filtros por região e comparativos de períodos.

---

## Entregáveis
- Notebook de auditoria  
- 6 arquivos `.csv` limpos  
- Notebook de análise  
- Scripts SQL  
- Painel Power BI  
- README.md
