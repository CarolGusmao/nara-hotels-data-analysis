# 📊 Projeto Final – NaraHoteis  

## 🎓 Sobre o Projeto  

Este projeto foi desenvolvido como **Trabalho de Conclusão do curso de Análise de Dados do SENAC**.

O objetivo foi aplicar, de forma prática, todo o processo de análise de dados, desde a auditoria e tratamento das bases até a construção de um dashboard gerencial para apoio à tomada de decisão.

A empresa fictícia **NaraHoteis** é uma rede com 12 unidades no estado do Rio de Janeiro. A diretoria identificou uma queda na receita e solicitou uma análise estruturada para entender os fatores que impactam o desempenho financeiro e operacional.

---

## 🎯 Objetivos  

- Realizar auditoria e tratamento de dados  
- Garantir padronização e qualidade das informações  
- Executar análise exploratória e estatística  
- Modelar banco de dados relacional  
- Criar dashboard estratégico no Power BI  
- Gerar insights para apoio à tomada de decisão  

---

## 🗂️ Bases de Dados  

Foram analisadas 6 bases principais:

- Clientes  
- Canais de venda  
- Reservas  
- Tipos de quarto  
- Unidades  
- Funcionários  

---

## 🔎 Etapas do Projeto  

### 1️⃣ Auditoria e Tratamento de Dados  

- Verificação de inconsistências  
- Tratamento de valores nulos  
- Padronização de formatos  
- Correção de erros de digitação  
- Exportação das bases limpas em `.csv`  

---

### 2️⃣ Análise Exploratória e Estatística  

Realizada em Python, com foco em responder perguntas estratégicas da diretoria:

- Como está o faturamento por unidade?  
- Qual a taxa de ocupação?  
- Quais canais geram mais receita?  
- Existe relação entre avaliação e receita?  

Foram utilizados gráficos e métricas para identificar padrões e oportunidades de melhoria.

---

### 3️⃣ Modelagem e Banco de Dados  

- Criação do banco `narahoteis_db` no MySQL  
- Definição de tabelas e relacionamentos  
- Implementação de chaves primárias e estrangeiras  
- Consultas utilizando `JOIN`  

---

### 4️⃣ Dashboard no Power BI  

- Conexão com os arquivos tratados  
- Criação de KPIs estratégicos  
- Indicadores de faturamento e ocupação  
- Comparativo entre períodos  
- Filtros por unidade e região  

---
